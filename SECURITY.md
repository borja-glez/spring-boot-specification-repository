# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please follow these steps:

1. **Do NOT open a public issue.** Security issues can be reported via [GitHub Security Advisories](https://github.com/borja-glez/spring-boot-specification-repository/security/advisories/new).
2. Include a detailed description of the vulnerability, steps to reproduce, and potential impact.
3. Allow reasonable time for the maintainers to respond and address the issue before any public disclosure.

We will acknowledge receipt of your report within 48 hours and aim to provide a fix or mitigation plan within 30 days.

Thank you for helping keep this project and its users safe.
